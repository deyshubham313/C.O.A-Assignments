🧠 Important Formula

Average Memory Access Time (AMAT):

AMAT=Hit Time+(Miss Rate×Miss Penalty)
⚙️ Summary of I/O Methods
Programmed I/O: CPU continuously checks the device status → simple but wastes CPU time
Interrupt-driven I/O: CPU performs other tasks and responds only when interrupted → better efficiency
DMA: Data transfer occurs directly between memory and device → highest performance
